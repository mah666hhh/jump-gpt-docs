Jump GPT - プライバシーポリシー

1. データ収集について

Jump GPT拡張機能は、ユーザーの個人情報を一切収集しません。

2. テキスト選択データの取り扱い

ユーザーが選択したテキストは、ChatGPTへの送信にのみ使用され、一時的なメモリ上でのみ保持されます。選択テキストは永続的に保存されることはありません。

3. 第三者へのデータ提供

当拡張機能は、選択されたテキストをChatGPT（OpenAI）に送信する以外、いかなる第三者にもデータを提供しません。

4. 必要な権限について

当拡張機能は以下の権限のみを使用します：

  • activeTab: テキスト選択の検知のため
  • scripting: ChatGPTへのテキスト送信のため
  • tabs: 新規ウィンドウでのChatGPT表示のため
  • system.display: ウィンドウ位置の最適化のため
  • contextMenus: 右クリックメニューの提供のため

5. お問い合わせ

本プライバシーポリシーに関するお問い合わせは、Chrome Web Storeの開発者連絡先までお願いいたします。

6. プライバシーポリシーの更新

本プライバシーポリシーの内容は、必要に応じて更新される場合があります。重要な変更がある場合は、Chrome Web Store上で告知いたします。
